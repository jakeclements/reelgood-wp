Responsive Video WordPress Widget Readme

  • Author - Kirsty Burgoine
  • URL - http://www.kirstyburgoine.co.uk/news/responsive-video-wordpress-plugin-released/

Requirements

  • WordPress 3.0 and above with WordPress Widgets

Installation

  1. Download responsive-video.zip
  2. Upload responsive-video.zip to the plugins directory
  3. Enable Responsive Video in the Plugin admin panel
  4. For videos in widget areas. In widget admin panel, place Responsive Video in the sidebar, and edit it to enter the YouTube / Vimeo URL, not the embed code
